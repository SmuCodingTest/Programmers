> 프로그래머스 스터디 

# 스터디규칙 </br>
### 문제 풀이
1. 주마다 1문제씩 선정해서 문제 풀이를 진행 
   - 개념 + 알고리즘 + 풀이방식 노션에 정리 및 설명
   - ex) 회의전까지 다음주 문제를 선정해오고 풀어오는 방식
2. 각자 branch를 만들어서 push 하기

### 진행 방식 </br>
1. 주차 과제를 해오지 않으면 다음주에 추가로 진행
2. 회의시간은 기본 1시간
